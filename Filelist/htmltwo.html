1.0          Introduction

1.1        Purpose

This document describes one of several tests that cover requirements for the LillyCharts iOS Framework.  Together, the tests verify the requirements listed in the LillyCharts iOS Framework Traceability Matrix as having a Verification Method of Manual Testing.

1.2        Scope

1.2.1       In-Scope

This document will cover:

-          The verification of LillyCharts iOS Framework requirements that are identified in the header of each test case.

1.2.2       Out-of-Scope

This document will not cover:

-          The verification of any requirements for the LillyCharts iOS Framework that are not listed in the test case header.

-          The verification of requirements for any other Framework or application.

1.3        Document Overview

Test Protocol Objective:

Verify the LillyCharts iOS Requirements that need to be tested manually.

Acceptance Criteria:

A test is considered to PASS if all expected results have passed.

A test is considered to PASS with Comment if passing the overall test script for situations where a non-critical test step fails and rationale has been approved that identifies it as a non-critical step.

A test is considered to FAIL if any expected result does not pass.

3.1         [Test10]  [Madelyne Progress Pie Chart  Chart Attributes]

 

Test Objective:

Verify that the Progress Pie Chart is displayed correctly on the screen.

Requirements addressed in this Test:

LIB_CHART-IOS_SRS-72

LIB_CHART-IOS_SRS-74

LIB_CHART-IOS_SRS-79

Test Device:

 

Operating System:

 

 

Test Setup:

-          The ChartViewSample application should be pre-loaded onto the test device.

Actual Result / Comments

Status

1

Launch the ChartViewSample application with the test device in portrait orientation

The ChartViewSample application is displayed in portrait orientation.

2

Select the Madelyne Progress Pie Chart entry.

Verify that the Madelyne Progress Pie Chart is displayed in portrait orientation.

Navigation Step.

 

 

3

Look at the progress pie chart on the screen.

Verify that the progress pie chart appears as a ring.
